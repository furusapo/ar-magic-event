# 魔法石 AR探索

スマホのブラウザで動く、AR.jsのマーカー型ARです。

## ファイル
- `index.html`：AR本体
- `marker.html`：HIROマーカー印刷ページ

## 試し方
1. GitHub PagesなどHTTPS環境に公開
2. iPhone/Androidで `index.html` を開く
3. カメラを許可
4. 別端末または印刷した `marker.html` のマーカーを映す
5. 魔法石が表示されたら「魔法石を獲得する」を押す

## 注意
- カメラ利用にはHTTPSが必要です（localhostを除く）。
- LINEやInstagram内ブラウザで動かない場合はSafari/Chromeで開いてください。
- 獲得記録は端末のlocalStorageに保存されます。
